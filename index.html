AI-assisteret refaktorering

fordi livet er for kort til manuel find-and-replace

~/project/.claude/
📁 .claude/
  📄 CLAUDE.md   # Projektkontekst & regler
  📁 skills/
    📄 component-redesign.md
    📄 migrate-dialog.md
    📄 fix-alerts.md
  📁 settings/
    📄 settings.json
/component-redesign

Et custom Claude Code skill der læser den gamle komponentkode, forstår DIMS API'et, og spytter den refaktorerede version ud — med korrekte props, imports og styling-migrering.

// I din terminal, kør bare:
$ claude
> /component-redesign src/components/OldButton.tsx
1
Claude læser CLAUDE.md for projektkontekst & DIMS-konventioner
2
Skill-skabelonen fortæller Claude hvordan der refaktoreres: prop mapping, import paths, patterns
3
Claude omskriver komponenten, kører npm run lint, og viser dig diff'en
4
Du reviewer, godkender og committer. Færdigt på sekunder, ikke timer.
